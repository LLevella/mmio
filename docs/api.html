API Notes

Matrix Types

  • mmio::CooMatrix<T>
  • mmio::CsrMatrix<T>
  • mmio::CscMatrix<T>
  • mmio::DenseMatrix<T>

Readers

  • ReadMatrixMarket<T>(stream)
  • ReadCSRMatrixMarket<T>(stream)
  • ReadCSCMatrixMarket<T>(stream)
  • ReadDenseMatrixMarket<T>(stream)
  • ReadCSRMatrixMarketFileStreamed<T>(path)
  • ReadCSCMatrixMarketFileStreamed<T>(path)

Writers

WriteMatrixMarket(stream, matrix) supports COO, CSR, CSC, and dense matrices.

Options

  • ReadOptions: symmetry expansion, duplicate handling, sparse ordering.
  • WriteOptions: header, duplicate handling, symmetry validation, triangular output.
  • DuplicatePolicy::sum and DuplicatePolicy::keep.
  • SparseOrdering::sorted and SparseOrdering::preserve.
