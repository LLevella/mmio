# Security Policy

`mmio` parses text Matrix Market files and does not execute file contents.
Security-sensitive reports should still be handled privately when possible.

## Supported Versions

The latest stable release line is supported.

## Reporting a Vulnerability

Please report suspected vulnerabilities through GitHub private vulnerability
reporting if it is enabled for the repository, or by contacting the maintainer
privately.

Useful details include:

- affected version or commit
- compiler and platform
- input file or minimal reproducer
- observed behavior and expected behavior
